## 📖 项目简介

本项目旨在收集和整理 **LeetCode Hot 100** 题目的详细解法，提供多种编程语言的实现，帮助开发者系统性地学习和掌握算法与数据结构。

每道题目都包含：
- 📝 详细的问题描述
- 💡 清晰的解题思路
- 💻 多语言代码实现（Python、Java、C++ 等）
- ⏱️ 时间和空间复杂度分析
- 🔗 相关题目推荐

## 🚀 快速开始

### 浏览题解

1. 在 [题目列表](#📋-题目列表) 中找到你想学习的题目
2. 点击题解链接查看详细内容
3. 参考代码实现和思路分析

### 贡献题解

我们欢迎所有形式的贡献！如果你想添加新的题解或改进现有内容：

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/add-solution`)
3. 参考 [TEMPLATE.md](TEMPLATE.md) 编写题解
4. 按照 [CONTRIBUTING.md](CONTRIBUTING.md) 的规范提交代码
5. 提交 Pull Request

## 💡 使用建议

### 对于初学者
- 从简单题目开始，逐步提升难度
- 先自己思考，再查看题解
- 尝试用不同方法解决同一问题
- 理解时间和空间复杂度的重要性

### 对于进阶者
- 关注代码的优化和边界条件处理
- 学习不同解法的优缺点
- 总结常见的算法模板和技巧
- 尝试举一反三，解决相关题目

## 📊 学习路线

推荐的学习顺序：

1. **基础数据结构**：数组、字符串、链表、栈、队列
2. **基础算法**：双指针、滑动窗口、哈希表
3. **进阶数据结构**：树、堆、图
4. **进阶算法**：二分查找、DFS/BFS、回溯
5. **高级算法**：动态规划、贪心、位运算
